# Terraform Lab 3

## Мета
Розгортання ізольованого середовища в GCP з використанням Terraform.

## Передумови (Prerequisites)
- Встановлений Terraform (v1.14+)
- Встановлений Google Cloud SDK (gcloud)
- JSON ключ Service Account для GCP
- Активний проект у GCP

## Використання Terraform
- Ініціалізація: `terraform init`
- Планування: `terraform plan`
- Застосування: `terraform apply`
- Видалення інфраструктури: `terraform destroy`
